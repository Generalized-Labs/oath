# Legal and compliance readiness

The following are blocking external approvals, not engineering checkboxes:

| Deliverable | Current state | GA requirement |
| --- | --- | --- |
| Terms of Service | Not effective | Counsel-approved and published |
| Privacy Policy | Not effective | Counsel-approved, data map verified |
| Data Processing Addendum | Not effective | Counsel-approved with SCCs where applicable |
| Subprocessor list | Not published | Vendor inventory, notice process, public page |
| Enterprise SLA | Draft objectives only | Approved remedy, exclusions, support obligations |
| SOC 2 | Controls not audited | Type I readiness decision; Type II observation started |
| Security review | Internal only | Independent architecture review and penetration test |

No repository contributor may mark these gates complete based only on drafted
text. Completion requires the approved document, effective date, responsible
owner, and public or customer-accessible location in the signed GA evidence
manifest.
