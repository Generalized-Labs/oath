# Support policy

Oath `v0.2.x` is a developer preview. Public usage questions and reproducible
bugs belong in GitHub Issues. Include the Oath version/commit, operating system,
Node/npm versions, command, expected result, actual result, and sanitized logs.

Do not put vulnerabilities, private package names, credentials, customer data,
or unpublished exploit details in a public issue. Follow `SECURITY.md` and email
`security@generalizedlabs.com` for security reports.

Preview support is best effort and has no response-time or availability SLA.
Design partners receive a named owner and escalation path defined in their
written beta agreement. A paid SLA is not effective until the counsel-approved
terms, support hours, severity definitions, remedies, and public effective date
are linked from the signed GA evidence manifest.
